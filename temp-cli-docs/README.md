## Quick Start

1. **Install the CLI**: See [Getting Started](./01-getting-started.md#installation)
2. **Initialize your project**: Run `rulebricks init`
3. **Deploy**: Run `rulebricks deploy`
4. **Check status**: Run `rulebricks status`
